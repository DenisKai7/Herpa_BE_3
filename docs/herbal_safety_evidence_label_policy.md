# Herbal Safety/Evidence Label Policy

Safety:
- `safe` → Relatif aman
- `limited` → Data keamanan terbatas
- `caution` → Perlu perhatian
- `unsafe` → Tidak disarankan
- `unknown` → Data keamanan belum cukup

Evidence:
- sources → Data sumber tersedia
- claims → Data klaim tersedia
- traditional uses → Data tradisional tersedia
- none → Data bukti belum tersedia

Do not display verified source labels when sources are empty.
